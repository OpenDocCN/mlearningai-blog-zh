## 如何获得股票市场数据和创建可视化(烛台，OHLC，PnF 等)。)使用 mplfinance 库

[Mplfinance](https://github.com/matplotlib/mplfinance) 是来自 [matlplotlib](https://matplotlib.org/) 的专用数据可视化软件包，用于可视化财务数据。
Mplfinance 能够以不同类型的图表形式可视化财务数据——蜡烛图、伦科图、OHLC 图、点和图形图等。只有很少几行代码。
