Azure Data Studio 是一个连接到 SQL 数据库的环境，包括一系列功能。Data Studio 能够为 SQL 实用程序和分析提供低代码体验。

> Azure Data Studio 提供了一种现代的、以键盘为中心的 **SQL 编码**体验，通过内置功能(如多个选项卡)使您的日常任务变得更加简单
