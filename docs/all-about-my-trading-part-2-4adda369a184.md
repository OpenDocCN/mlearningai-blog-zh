# 经纪人级别的市场数据

外汇市场的一个有趣的特点是它没有集中的交易所，如股票或期货交易所。因此，所有交易都是在柜台(OTC)进行的，这仅仅意味着它不是在受监管的环境中进行的，实际上通常被称为“场外”交易。
