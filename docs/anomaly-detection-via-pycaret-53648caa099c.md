# 介绍

[**异常检测**](https://en.wikipedia.org/wiki/Anomaly_detection) ，或**异常值分析**，是数据科学工作流程中的一个步骤，用于识别偏离标准、正常
或预期的数据点。

> 异常值是指与其他观测值偏离太多，以至于引起怀疑的观测值
