# 我是谁

> 2016 年放学后，在德国伍珀塔尔学习商业数学。在前两年，我发现了自己对机器和深度学习相关主题的热情。

这就是为什么我在学习的同时开始编码和开发新的技术技能。在我的第三年，我很好奇在学术世界之外获得经验，所以我申请了我的第一次实习。

从那时起，在过去的四年里，我担任过实习生或勤工俭学学生的多个职位，我想在下面分享更多细节。

自今年 10 月以来，我现在是德国慕尼黑一家初创公司的全职数据科学家，致力于处理从加速度和回转数据中检测异常的问题，以检测汽车中的车辆损坏。

# 我的第一份数据分析师实习

> 毕业后，我并不真正知道自己想做什么，我开始简单地寻找适合我兴趣的公司或更具体的职位。

当时，它主要是金融相关主题的组合，我可以将我的应用数学知识应用到其中。这就是为什么我的第一份申请去了我家乡附近的大公司。

幸运的是，在我寄出的四份申请中，有一份非常有前途。Postbank AG 数字销售部门的数据分析师职位。

虽然我今天会以不同的方式申请，但在我第一次实习之前，我只是简单地填写了在线申请表。幸运的是，在经历了一个在我看来非常公平的两轮面试后，我得到了这家银行的机会。

关于面试的准备，我主要是对公司本身进行研究，而不是太关注我的技能，这是我现在会做的不同的事情。

# **我的第一份学生工作是 ML 工程师**

在我第一次实习和本科学习快结束时，指导我论文的教授告诉我一个计算机视觉工程师的工作职位。

这个机会来的正是时候。该公司名为 ControlExpert GmbH，是一家中型公司，专注于处理车祸保险索赔，在 R&D 设有一个大型部门。

> 与一家公司的直接接触几乎保证了我至少能得到一次现场面试。只有一封附有我简历的电子邮件就足够了。

知道我被推荐后，我非常自信地去参加面试，特别是因为我自己已经完成了几个可以谈论的项目。

> 回想起来，我相信我能够详细谈论我自己完成的项目的信心是获得这份工作的决定性因素。之前的实习经历和一位教授的推荐当然也帮了我很多。

# **我的第二份实习是开发 NLP 算法**

由于 COVID，我很不幸地从我最近的学生工作中被解雇了。由于很难找到新的机会，我在研究生的第一年主要致力于顺利通过我的课程。

> 现在在 LinkedIn 和 GitHub 等平台上更加活跃，似乎让我对自己产生了一点兴趣。

不断地在 GitHub 上创建新的存储库，并用它记录我的学习进度，产生了这样的效果，Infosys 有限公司的招聘人员联系了我。

> 同样，我的准备主要集中在我的项目和轻松谈论项目细节的能力上。此外，我还练习用英语谈论和解释与工作相关的话题。

在第一次联系后，我们很快安排了与招聘人员的视频通话，这就像是一次关于我的兴趣、经历和期望的愉快聊天。

几天之内，我已经收到了回复，并进行了后续采访(由于 COVID 也在偏远地区)。这次是两位数据科学家。尽管质量很糟糕，而且这是我第一次用英语进行的技术面试，但看起来进行得还不错。

# **我最近的学生工作是数据科学家**

甚至在我之前的实习开始之前，我的一个老同事就在慕尼黑一家新成立的公司给我发布了一份工作，从事从加速度和回转传感器自动检测汽车损坏的工作。

在那之前我已经实习过很多次了，知道我还很年轻就能成为一名有价值的团队成员，这无疑增强了我的信心。

> 此外，进行了多次采访后，我现在能够谈论我的大多数项目，甚至不需要准备。

有趣的是，与初创公司 CTO 的第一次通话甚至不到半个小时，似乎他们几乎了解我的一切，这就是为什么面试感觉像是一次轻松的聊天。

因为我已经签了印孚瑟斯实习的合同，所以我协调了之后的会面。因为我已经感觉到这份工作相当稳定，所以我没有花太多时间准备。

总而言之，回想起来，我的直觉是正确的，这也是我最终在 carValoo 找到工作的原因。

有人推荐会让整个过程变得简单很多。

到今天为止，我已经在那里工作了一年多，从一名学生变成了一名全职的远程员工。
