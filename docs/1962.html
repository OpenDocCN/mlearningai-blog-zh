在Data Studio中使用函数

Azure Data Studio是一个连接到SQL数据库的环境，包括一系列功能。Data Studio能够为SQL实用程序和分析提供低代码体验。

Azure Data Studio提供了一种现代的、以键盘为中心的 SQL编码体验，通过内置功能(如多个选项卡)使您的日常任务变得更加简单
