由于机器学习在理解文本方面的强大能力，它已经成为文本分类的一个组成部分。文本分类是将文本数据分类到各种预定义的类别并相应地标记它们的过程。机器学习算法用于自动分类大量未标记的文本，使人类更容易识别相似文本中的相同类别
