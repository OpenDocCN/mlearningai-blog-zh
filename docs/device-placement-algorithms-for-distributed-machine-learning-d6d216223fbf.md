近年来，对训练神经网络的计算要求有很大的需求。在分布式训练环境中，神经网络被分割在多个 GPU 和 CPU 设备上。分布式训练面临的挑战是找到一种放置多个异构设备的优化方法，以实现最快的训练速度。
