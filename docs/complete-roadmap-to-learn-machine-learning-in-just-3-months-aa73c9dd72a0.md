# 如何学习机器学习？

在告诉你你需要学习的基础知识之前，我想说你应该按照我在这里从上到下的顺序遵循这条道路

因为，当你从上到下理解时，你会很舒服地把所有的概念都搞清楚，因为有些概念是相互关联的

所以，只要记住这一点，现在让我们进入我们的主题，以获得路线图

## 像 Python 和 R 这样的编程语言:

![](img/da4adc5aca3c9eaa02f6492731405a2d.png)

Photo by [Chris Ried](https://unsplash.com/@cdr6934?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

你必须学习的第一件事就是编程语言

因为学习编程语言在机器学习或数据科学领域非常重要

因为，无论你将进一步构建什么样的模型，你都将使用编程语言

为此，你需要学习编程语言，在这里你不需要像软件开发人员那样掌握它

我的意思是，你需要学习编程语言中所有的基本概念，你也可以轻松地学习一些中间的东西

但是，你至少应该知道如何编码和一些基本的数据结构和面向对象的编程概念

现在，你可能会问，我们需要同时学习 Python 和 R 吗？

那么我会说没有，因为大多数公司都在使用 python，所以你可以学习 Python

但是，如果你想学习这两种语言，那么你可以学习，因为擅长这两种语言对你有好处

一旦你学会了 Python，那么学习 R 语言就很容易了，因为你已经掌握了编程中使用的语言的基本概念

但是你必须学习一种编程语言，你可以学习 Python 或者 R

## 不同的库:

现在，在学习 Python 或 R 之后，接下来的事情是学习数据科学或机器学习领域的不同库

因为这些库将帮助你理解你将用来建立不同的机器学习模型的数据

> 还有一些不同的库，像 Numpy、Pandas、Matplotlib、Sklearn 等，它们在这个领域非常重要，所以，只要努力学习所有这些库并进行练习，这样你就会熟悉它们，并可以进一步使用它们

## 探索性数据分析:

现在读完这个词后，如果你感到困惑，如果你有这样的想法，这种探索性的数据分析到底是什么？

那么不要担心，伙计们，因为只是单词有点复杂，但其中包含的东西是很容易的

所以，伙计们，探索性数据分析什么都不是，但你必须非常好地理解数据，这样你才能进一步利用你拥有的数据建立模型

如果我告诉你探索性数据分析中的基本概念，以及你应该如何分析数据，那么你需要遵循几件事情

例如，您必须查看数据中列的数据类型，您必须检查是否存在任何空值，然后您可以查看功能之间的关系

现在，这里的要素只是数据中的列，有两种类型的要素，一种是独立要素，另一种是从属要素

但是在这里，我不想深入地告诉你关于这方面的技术问题，但是一旦你开始学习这些东西，你就会遇到，并且你会在那个时候清除所有这些东西

但是现在，我只是告诉你在这个 EDA(探索性数据分析)部分你可以或将要学习的概要

因此，它的基本目标就是查看您拥有的全部数据，并尽可能多地分析这些数据

## 特征工程:

现在，在学习了探索性数据分析之后，你必须学习特征工程

因为，当您对数据集执行探索性数据分析时，下一步就是对该数据执行特征工程

在特征工程中，你必须学习一些重要的东西，这将帮助你在机器学习中建立更好的模型或项目

在探索性数据分析后的下一步功能工程中，我们执行一些任务，如处理缺失值、处理数据集中的异常值、分类编码、标准化以及数据的缩放等

这些是一些重要的术语，或者说是你在特征工程中需要学习的东西

因为这些东西会帮助你擅长机器学习，并在其中脱颖而出

## 功能选择:

现在，当您学习特征工程时，下一步是学习特征选择

顾名思义，您只需了解如何使用特征选择过程从数据集中选择正确的特征

在特征选择中，您需要学习一些东西，如相关性、单变量选择、向前排除、向后排除、决策树特征选择、随机森林重要性等

现在，不管我们已经学到了什么，探索性数据分析、特征工程和特征选择，这些步骤将会占用你 30%到 40%的时间

现在，我的意思是，当你建立一些项目时，你将致力于这三个主要因素，这将占用你整个项目时间的 30%到 40%

因为这三件事在你试图构建任何项目时都是非常重要的

所以，我会说，你必须正确地学习所有这些东西，以便在公司里，当你获得数据时，你能够以更正确的方式分析这些数据

## 机器学习算法—(回归和分类、聚类)

![](img/ec566df1489e952bca9b4580f6f6a760.png)

Photo by [Alex Knight](https://unsplash.com/@agk42?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

在学习完以上三件事情之后，现在我们的主要任务从这里开始，也就是学习机器学习中所有重要的算法

这里你需要学习像回归和分类、聚类等算法

在回归中，你将学习简单线性回归，多元线性回归，在分类中，你将学习逻辑回归

在聚类中有不同的算法，如 K 均值聚类、层次聚类等

聚类是一种无监督的机器学习算法，如果我们谈到回归和分类，那么这些都是有监督的机器学习算法

还有一些其他不同的算法也像随机森林，决策树，XGBoost 等是集成技术

并且这些算法可以具有回归和分类问题语句

学习完这些算法后，你会遇到朴素贝叶斯算法，这是一种分类

然后你有一些其他的算法，像支持向量机，这是用于回归和分类

一种是支持向量回归，另一种是支持向量分类，既用于回归又用于分类

正如我们已经讨论过的聚类，还有另一种算法，即 K 近邻聚类，您将在学习聚类算法时学习到这种算法

所以，这些是你在机器学习中需要学习的一些重要算法

## 超参数调谐:

现在，在学习了简单线性回归、多元线性回归、逻辑回归等分类算法等所有重要算法之后，您必须学习超参数调整

这种技术您将在使用线性回归或分类算法构建机器学习项目时使用

这在构建项目时是一件很重要的事情，因为这将花费你机器学习项目时间的 10%到 20%

因为使用这种技术，您将从数据集中提取最重要的特征，这将帮助您提高项目或模型的准确度

> 所以在超参数调优中，需要学习网格搜索 CV，随机搜索 CV，超视，Optuna 等

但我会说，如果你是一个初学者，如果你刚刚开始学习机器学习，那么我会说你应该擅长网格搜索 CV 和随机搜索 CV

因为这两个是在建立机器学习模型时最常用的算法

但是很多时候很多人尝试使用随机搜索 CV，因为它比网格搜索 CV 快

在学习超参数调谐时，您还可以学习一些不同的技术

## 不同的云平台:

当你将学习我们已经讨论过的所有上述内容时，之后你可以学习不同的云平台，这样它们将在这个领域帮助你

互联网上有许多不同的云平台，其中一些是亚马逊网络服务(AWS)、Azure、谷歌云等

模型部署:

你可以在不同的平台上部署你的机器学习项目或模型

你可以将它部署在 Heroku 或任何其他知名平台上，正如我所说，选择哪个平台取决于你
