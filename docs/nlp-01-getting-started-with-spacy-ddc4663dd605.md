**自然语言处理简介**

NLP 是人工智能的一个子领域，它分析文本、语音和其他形式的人类生成的语言数据。NLP 使计算机能够提取关键词和短语，理解语言的意图，将其翻译成另一种语言，或者生成响应。

NLP 可以用来创建搜索引擎算法，聊天机器人，语音识别应用程序…
