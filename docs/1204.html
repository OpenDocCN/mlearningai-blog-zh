Terra form是由hashi corp打造的开源基础设施as code (IaC)软件工具。它提供一致的CLI工作流来管理数百个云服务。Terraform将云API编码成声明性的配置文件。它读取配置文件并提供变更的执行计划，可以对该计划进行安全性审查，然后根据需求进行应用和配置。
