# 计算机编程语言

Python 是我拥有的第一个工具，也是让我掉进数据分析兔子洞的工具，它让我走到了今天。Python 最好的一点是它不需要任何下载费用。这引起了我的兴趣，因为任何对数据分析感兴趣但不知道从哪里开始的人都应该感兴趣。此外，它所拥有的支持网络令人难以置信。花费大量时间在堆栈溢出上寻找现代问题的解决方案，却在一篇八年前的文章中找到答案，这种情况并不罕见。超过 100，000 个图书馆正在打开一个充满机会的世界，让你看起来像一个数据魔术师，那么它对我有什么帮助呢？

自动化单调的工作，比如每天早上通过电子邮件发送报告，是我最有价值的举措之一。你不希望你守时的坏名声掩盖了你提供的价值。除了您节省的时间，领导也会很高兴知道自动化节省了他们部门的时间，这是所有领导都引以为豪的事情。

如果您每天都要处理大量的数据，而又不了解 Python，那么您将会错失良机。Excel 已经有几十年的历史了，大家都知道怎么用，但是在操作大型数据集的时候阻碍了速度慢的人。例如，Excel 中的 VLOOKUP 将比 Python 中合并两个大型数据集花费更长的时间。

# （舞台上由人扮的）静态画面

几乎每个人都在有规律地制作电子表格，将模糊的数据总结成易于理解的视觉效果，供领导层使用。如果我说你只需要造一次那个东西呢？这会节省你多少时间？我从在 Tableau 中重建电子表格并让它们按计划更新中受益匪浅。节省时间是 Tableau 的主要优势，所有内置功能使其成为用数据讲述故事的宝贵工具。

有了 Power BI 或者 Python 中的库，比如 [Bokeh](https://docs.bokeh.org/en/latest/docs/gallery.html) ，在当今世界没有 Tableau 也能生存。然而，你会错过 Tableau 提供的功能。Tableau 的计算帮助和图表建议可确保您充分利用数据。此外，它还有一个大规模的支持网络，提供[历史解决方案](https://community.tableau.com/s/question/0D54T00000C6HCnSAN/multiple-line-charts-5-lines-in-a-single-graph)，帮助您回答任何问题。

Tableau Server 在组织内蓬勃发展有多种原因。它将所有仪表板保存在一个安全的位置，并允许分析师将数据写入服务器供其他人访问。例如，如果您的工作需要编辑数据集，有人可以在部门服务器上发布它。此功能允许您访问编校数据以构建仪表板，同时不会有显示非公开数据的风险。

# Alteryx

如果您有幸在拥有 Alteryx 许可的组织中工作，您不应该错过学习这个工具的机会。Alteryx 通过允许分析师使用可视化工具和连接而不是代码行来构建工作流，简化了数据分析学习过程，这是一件好事，原因如下:

1.  用户界面不像看成千上万行 Python 代码那样令人望而生畏，所以它不太可能让初学者望而却步。
2.  它为构建者提供了一个可操作的工作流程，当他们需要解释他们是如何得到答案的时候，向观众展示。

Python 代码看起来比现实更具挑战性。看起来像外语的代码行或需要一定程度才能理解的概念都是造成 Python 误解的属性。结果，你最终会遇到继续使用错误工具的分析师。Alteryx 的图片工具几乎不可能被误解。

我们与许多个人合作，并开始识别他们作为我们数据接收者的细微差别。有些人喜欢收到一份包含他们所要求的所有 KPI 的汇总报告，而其他人则喜欢用底层数据来娱乐自己。了解你的听众并为他们的提问做好准备是至关重要的。如果我要为喜欢看开发过程但不了解 Python 的人制作一份报告，那么 Alteryx 将是更好的选择。它可以让你像仪表板一样讲故事，同时让听众参与进来。

Alteryx 还有一个服务器，允许组织将工作流保存在一个集中的位置，供每个人访问。没有必要花费时间重新发明轮子，因为员工通过复制工作流程的部分以在工作中使用来节省时间将成为一种常见的做法。如果你有兴趣阅读更多关于 Alteryx 的功能，那么你可以从 [Megan Dibble](https://medium.com/u/271578f29747) 那里了解更多。
