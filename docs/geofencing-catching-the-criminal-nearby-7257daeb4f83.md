# 什么是地理围栏？

地理围栏是一种使用 GPS 或射频识别(RFID)来创建虚拟地理边界的技术，使软件能够在移动设备进入或离开特定区域时触发响应。地理围栏可用于无害的营销目的，如发送基于位置的优惠券或提示，或用于更严重的安全…
