下面使用的数据集是在 [Tableau 社区](https://community.tableau.com/s/question/0D54T00000CWeX8SAL/sample-superstore-sales-excelxls)上提供的超级商店数据集。

*   将日期变量拖到列架上

![](img/bfbf9efbf894302867a41212eff94212.png)
