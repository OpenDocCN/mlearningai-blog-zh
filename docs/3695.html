介绍

机器学习在Python中被广泛编码。这对于培训来说是可以的，但是我们必须让我们的产品更快，并集成部署的应用程序。为此，我决定提供这方面的教程。希望这些教程对工程师和研究人员有所帮助。所有代码都可以在 Github repo 获得。如果你正在开发PyTorch，你也可以在c++中查看PyTorch的这个系列。

装置

我们将使用c++中的机器学习库 mlpack 库。要安装mlpack，您可以遵循以下步骤。

安装犰狳

sudo apt-get install liblapack-dev
sudo apt-get install libblas-dev
sudo apt-get install libboost-dev
sudo apt-get install libarmadillo-dev

安装ensmallen

git clone https://github.com/mlpack/ensmallen.git
cd ensmallen
mkdir build & cd build
cmake ..
sudo make install

安装谷物

sudo apt-get install libcereal-dev

安装mlpack

git clone https://github.com/mlpack/mlpack.git
cd mlpack
mkdir build && cd build
cmake ..
sudo make install

在这些步骤之后，我们可以在c++代码中包含mlpack.hpp。
