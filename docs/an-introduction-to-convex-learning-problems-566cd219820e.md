在本文中，我们将讨论*凸学习问题*。原因是凸学习是一个重要的学习问题家族，因为我们能够有效学习的大部分内容都属于这一类别。因此，在本文中，我们将定义什么是一个*凸学习问题*。要做到这一点，我们需要定义什么是'*凸性*'。除此之外，我们还将定义诸如' *Lipschitzness* 和'*smooth*的概念，因为当我们定义*凸学习问题*时也需要它们。
